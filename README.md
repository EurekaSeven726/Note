### 标识符的命名规则

---

• 标识符可以由字母、数字、下划线（_）和美元符($）组成，不能以数字开头
• ==标识符严格区分大小写==
• 标识符不能是Java关键字和保留字
• 标识符的命名最好能反映出其作用
